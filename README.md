# GolfSwingWatch

Apple Watch golf swing analysis platform.

## Vision
Use Apple Watch motion sensors to identify, classify, and improve golf swings through data-driven feedback.

## Current Status
Phase 1 - Motion Capture MVP

## Core Technologies
- Swift
- SwiftUI
- watchOS
- CoreMotion
- SwiftData
- WatchConnectivity

## Analysis Stack (Starter)
- Python
- pandas
- numpy
- pyarrow (Parquet I/O)

See [DATA_PIPELINE.md](DATA_PIPELINE.md) for the full Watch → iPhone → Mac → dashboard workflow.
See `analysis/README.md` for the local feature extraction pipeline.
